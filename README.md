# Spatial sampling

University assignment exploring stratified cluster sampling on soil-chemistry data. Uses KMeans clustering with primary/secondary sampling units (PSU/SSU) across strata.

## Files

- `main.py` — main sampling pipeline
- `calculation_details.py` — sampling calculations
- `plot_cluster.py` — figure generation
- `main_manual_calculations.py` — manual verification of the sampling math
- `toy_dataset.py` — synthetic dataset for testing
- `soildata.csv` — input dataset
- Result figures: `cu_heatmap.pdf`, `selected_clusters*.pdf`, `soil_data_clustering.pdf`
